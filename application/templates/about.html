How it Works

This Bangalore Home Price Prediction application uses a machine learning model trained on thousands of real estate listings across Bangalore. The model uses Linear Regression to predict home prices based on:

  • Location (240+ areas in Bangalore)
  • Total square feet area
  • Number of bedrooms (BHK)
  • Number of bathrooms

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Flask (Python)
  • Machine Learning: Scikit-learn (Linear Regression)
  • Data Processing: Pandas, NumPy

Data Sources

Our model is trained on a comprehensive dataset of Bangalore real estate listings, covering properties across the city. The data includes information on property size, location, amenities, and historical sale prices.
